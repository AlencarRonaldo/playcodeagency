## 🚀 Como Executar

### 📋 **Pré-requisitos**
```bash
Node.js 18+ 
npm ou yarn
Git
```

### ⚙️ **Instalação**
```bash
# Clone o repositório
git clone https://github.com/AlencarRonaldo/playcodeagency.git

# Entre no diretório
cd playcodeagency

# Instale as dependências
npm install

# Execute em modo desenvolvimento
npm run dev

# Acesse http://localhost:3000
```

### 🏗️ **Build para Produção**
```bash
# Build otimizado
npm run build

# Executar produção local
npm run start

# Lint & verificações
npm run lint
```
